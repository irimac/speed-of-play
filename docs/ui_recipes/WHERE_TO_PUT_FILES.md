# Where to put the artifacts (recommended)

Copy into your repo:

- `assets/ui_recipes/`  (tokens.yaml, components.yaml, screens/*.yaml)
- `docs/ui_recipes/AGENT_CONTRACT_UI_RECIPES.md`  (this contract)
- `docs/ui_recipes/AGENT_PROMPT_TEMPLATE.md`      (agent prompt)

Optional but helpful:
- `mocks/derived_svgs/` (offline SVG previews generated from the recipes)
